# Nanoemulsion & PCM Research
This repository contains research work related to nanoemulsions and phase change materials (PCMs) in polymer and coating systems.
## Areas of focus
- Nanoemulsion formulation
- Phase Change Materials (PCM)
- Rheology analysis
- DLS (Dynamic Light Scattering)
- DSC (Differential Scanning Calorimetry)
- Zeta potential analysis
## Purpose
This project is part of my MSc research in Chemical Engineering at University of Mazandaran.
